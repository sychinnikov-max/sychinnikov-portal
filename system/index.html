01 AI-команда — 4 субагента

пересобрана 2026-05-18 · trajectory-architect добавлен 2026-05-21 · последняя сверка 2026-05-25

mentor

Главный наставник по жизненной траектории

Зачем
Стратегический советчик по долгосрочным целям $10М+ к 40. Задаёт вопросы и подсвечивает противоречия.
Когда
Цели 90 дней, главные развилки, семья и эмиграция в стратегическом разрезе.
Не для
Операционка, продажи, финрасчёты — есть специализированные.
trajectory-architect

Жёсткий наставник по упаковке Founder Trajectory

Зачем
Гибридный путь 6 недель — методология → продуктизация → позиционирование. Школа Бейкера + Эннса + Вайс.
Когда
Упаковка продукта, методология, цены, отстройка, фильтр клиента, мета-блок «импостер в комфорте оператора».
Не для
Жизненная стратегия (это mentor), продажа, контент.
finance-director

Семейный и проектный финдиректор

Зачем
Cashflow, runway, долги, инвестиции, equity-портфель.
Когда
Работа с деньгами — личными или клиентскими. Расчёты, бюджетирование, проверки.
Не для
Жизненная стратегия. Финансовые цели уровня «зачем» — это mentor.
octo-advisor

Внутренний советник проекта Octo

Зачем
Изолирован от остальной AndreiOS. Паттерн advisor — при долгосрочном клиенте создаётся свой <name>-advisor.
Когда
Стратегические разборы Octo, аудит проекта, помощь со скоупом.
Не для
Другие клиенты — у них свои advisor или общий контекст.
Правило вызова

Mentor задаёт рамку. Trajectory-architect исполняет внутри (упаковка продукта). Finance-director — деньги отдельным контуром. Octo-advisor — изолирован к одному проекту. Не путать функции.

03 Правила контекста и роста

экономия токенов · защита от раздувания

Правило роста
Новая папка/файл создаётся только когда: (а) действующий файл >200 строк, или (б) ≥3 однотипных артефакта шумят в общем, или (в) явно решено «нужно для одной из трёх целей 90 дней». Запрещено создавать «на всякий случай» или «для красоты».
Не читать archive/
Никогда не читать архив без явного запроса. Старые raw-файлы не переписывать, только архивировать.
Inbox только в /day-close
inbox/today.md — живой накопитель сырья. Читается только при разборе дня. При обычной работе — не загружать в контекст.
Contacts только по запросу
Картотека читается только при «порекомендуй человека для X», «кто у меня есть по теме Y». При обычной работе — не сканировать. contacts/_private/ исключается из автоматических рекомендаций.
Granola — отдельный источник
Pull только при /day-open и /call. В обычной работе не вытягивать — съедает токены. MCP подключён локально.
Pre-flight confidentiality
Перед любым sales/outreach: фильтр (а) Fishok-full-time-фрейм, (б) Octo-пауза. Контакт в красной зоне — в «запрещённые каналы», не в outreach.

05 Kill-switch

⚠ Если 14 дней подряд нет коммитов в репозитории — система признана мёртвой. Не строим следующую итерацию ещё 30 дней. Сначала ретроспектива «почему не работало», потом решение.

06 Активные режимы

Cashflow-only — с 2026-05-18

До момента когда чистый доход стабильно держится €8К/мес 2 месяца подряд — equity-обсуждение НЕ обязательный фильтр при входе в проект. Беру платные проекты без принудительного equity-разговора.

Equity обсуждаю только если: (а) клиент сам поднимает, (б) проект явно проходит все 4 жёстких критерия. После €8К/мес × 2 — обязательный equity-фильтр возвращается на любой проект >€2К/мес.

07 Что НЕ делать

накопленные уроки

  • Не предлагать LinkedIn-каналы привлечения до июня
  • Не холодные продажи через РФ-нетворки
  • Не упоминать Оскара / Маддеру / Татьяну в сообщениях Кристапсу
  • Не работать «no win / no fee» без письменной фиксации скоупа и фикса (урок Octo)
  • Не подписывать с founder-каналами модель «агент платит процент»
  • Не покупать технику и необязательные вещи до стабильного €8К+/мес
  • Не ставить задачи на выходные (договорённость с Викой)
  • Не создавать новых файлов/папок без триггера правила роста
  • Не делать утренних 30-минутных ритуалов — есть вечерний /day-close
  • Не показывать кассовый разрыв при асимметрии силы (урок Сергей-21.05)
