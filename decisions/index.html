Что и почему менялось — стратегические решения сверху-новые.

Append-only журнал. Решения по системе AndreiOS, продукту Founder Trajectory, финансам, клиентским проектам. Источник: decisions.md в Claude-Context + проектные ADR.

2026-05-25 founder-trajectory

Reset двухуровневый: v1 (3 мес, 1 преемник) → v2 (6 мес, премиум или equity)

v1 = 3 мес, 1 показатель, 1 преемник изнутри клиента. v2 = 6 мес, 2-3 показателя, нас двое. На v2 — премиум-цена ИЛИ equity. Reset v1 ценовой коридор €2 500-4 000/мес — защищаемый. Источник: pricing-ladder.md v1.
2026-05-25 founder-trajectory

Преемник изнутри клиента — обязательное условие продажи Reset

Без идентифицированного преемника внутри команды клиента — Reset не продаём. Тест уходом = финальная встреча как двухслойный экзамен: преемник проводит → Андрей зритель. Это маркер «актив-система остаётся работать».
2026-05-25 founder-trajectory

Метафора-метод: «врач системы» + ИКЕА

Врач системы: анамнез → анализы → диагноз → лечение. ИКЕА: диагноз + воспроизводимая система после выхода. Закрытая метафора для продукта и коммуникации.
2026-05-25 founder-trajectory

Equity-разговор стартует только на Reset v2, не на v1

На v1 — только фикс/retainer. Equity обсуждается только когда v1 уже отработан успешно, переход в v2. Связано с cashflow-only режимом и 4-критериями equity.
2026-05-25 founder-trajectory

BPMN 2.0 — обязательный формат описания инструментов

Все процессные инструменты Founder Trajectory описываются в нотации BPMN 2.0. Импорт в bpmn.io / Camunda / draw.io. Скилл /bpmn-generator. Унифицирует артефакты, отстраивает от инфоцыган.
2026-05-25 founder-trajectory

Метрика Reset = количество управленческих решений собственника на финальной встрече

Не «процент изменений», не «вырост выручки». Метрика — сколько решений собственник принимает на финальной встрече (которую ведёт преемник, Андрей зритель). Это операционный proxy того, что система работает без Андрея.

Якорь Fishok €3 500/мес и зарплата Татьяны как ФД отклонены. Якорь = рыночные данные (vChief, Гончаров-fractional-COO, Нескучные Финансы, market-niche-analysis-2026-05-25). >100 источников, baseline на 6-9 мес.
