# sychinnikov.com

Личный портал Андрея Сычинникова. Статические HTML-страницы, деплой через Cloudflare Pages.

## Структура

- `index.html` — главная с навигацией
- `articles/` — отдельные материалы (HTML)
- `robots.txt`, `sitemap.xml`, `llms.txt` — мета для поисковиков и ИИ-агентов

## Деплой

Любой push в `main` автоматически обновляет сайт через Cloudflare Pages.
